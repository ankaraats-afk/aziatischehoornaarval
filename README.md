# Aziatische hoornaar website - GitHub Pages

Dit pakket is klaar voor GitHub Pages als statische website.

## Inhoud
- `index.html` — de website
- `.nojekyll` — voorkomt ongewenste Jekyll-verwerking op GitHub Pages

## Uploaden naar GitHub Pages
1. Maak een repository op GitHub.
2. Upload `index.html` en `.nojekyll` naar de root van de repository.
3. Ga naar **Settings** > **Pages**.
4. Kies **Deploy from a branch**.
5. Kies branch **main** en folder **/(root)**.
6. Wacht even tot de site gepubliceerd is.

## Opmerkingen
- Voor een gewone projectsite mag de repositorynaam vrij gekozen worden.
- Voor een gebruikerssite moet de repositorynaam exact `<gebruikersnaam>.github.io` zijn.
- Sommige afbeeldingen laden van externe bronnen. Als je dat niet wilt, vervang die later door lokale bestanden in een map zoals `assets/`.
